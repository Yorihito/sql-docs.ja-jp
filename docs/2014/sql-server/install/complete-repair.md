# <a name="complete---repair"></a>[完了 - 修復]
  **[完了 - 修復]** ページを使用して、修復が完了したことを確認します。  
  
## <a name="options"></a>オプション  
 このダイアログ ボックスには、この操作のセットアップ ログ ファイルへのリンクが含まれています。 続行するには、 **[閉じる]** をクリックします。
