# <a name="sql-server-integration-services-100-sql-server-configuration-manager"></a>SQL Server Integration Services 10.0 (SQL Server 構成マネージャー)

 [!INCLUDE[ssIS](../../includes/ssis-md.md)]の詳細については、オンライン ブックを参照してください。
