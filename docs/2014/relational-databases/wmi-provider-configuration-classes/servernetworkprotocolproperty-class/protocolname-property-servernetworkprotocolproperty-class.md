# <a name="protocolname-property-servernetworkprotocolproperty-class"></a>ProtocolName プロパティ (ServerNetworkProtocolProperty クラス)
  参照されたプロパティを所有するプロトコルの名前を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>指定項目  
 *object*  
 のインスタンス上のネットワークプロトコルの属性を表す[Servernetworkprotocolproperty クラス](servernetworkprotocolproperty-class.md)オブジェクト [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
## <a name="property-valuereturn-value"></a>プロパティ値/戻り値  
 プロパティを所有するプロトコルの名前を指定する文字列値。
