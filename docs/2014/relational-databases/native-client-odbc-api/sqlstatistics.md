# <a name="sqlstatistics"></a>SQLStatistics
  **Sqlstatistics**は、静的カーソルで実行できます。 更新可能な (キーセットドリブンまたは動的) で**Sqlstatistics**を実行しようとすると、カーソルの種類が変更されたことを示す SQL_SUCCESS_WITH_INFO が返されます。
