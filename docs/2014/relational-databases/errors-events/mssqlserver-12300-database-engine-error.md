# <a name="mssqlserver_12300"></a>MSSQLSERVER_12300
    
## <a name="details"></a>詳細  
  
|属性|値|  
|-|-|  
|製品名|SQL Server|  
|イベント ID|12300|  
|イベント ソース|MSSQLSERVER|  
|コンポーネント|SQLEngine|  
|シンボル名|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|メッセージ テキスト|計算列は '*construct*' でサポートされていません。|  
  
## <a name="user-action"></a>ユーザーの操作  
 計算列は、メモリ最適化テーブルでサポートされていません。
