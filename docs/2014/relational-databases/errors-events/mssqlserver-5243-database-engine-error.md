# <a name="mssqlserver_5243"></a>MSSQLSERVER_5243

|属性|値|  
|-|-|  
|製品名|SQL Server|  
|イベント ID|5243|  
|イベント ソース|MSSQLSERVER|  
|コンポーネント|SQLEngine|  
|シンボル名||  
|メッセージ テキスト|内部操作中に一貫性が損なわれていることが検出されました。 テクニカル サポートにお問い合わせください。 参照番号 %ld。|
