# <a name="mssqlserver_12302"></a>MSSQLSERVER_12302
    
## <a name="details"></a>詳細  
  
|属性|値|  
|-|-|  
|製品名|SQL Server|  
|イベント ID|12302|  
|イベント ソース|MSSQLSERVER|  
|コンポーネント|SQLEngine|  
|シンボル名|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|メッセージ テキスト|PRIMARY KEY 制約の一部である列の更新は '*construct*' でサポートされていません。|  
  
## <a name="user-action"></a>ユーザーの操作  
 主キー制約の一部である列は更新しないでください。
