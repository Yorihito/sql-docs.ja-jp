# <a name="default-subkey"></a>既定のサブキー
レジストリでは、既定のサブキーを持つ既定のデータソースを指定できます。 このサブキーは、データソース指定サブキーの特殊なケースであり、他のデータソース仕様サブキーと同じ値を持ちます。 唯一の違いは、ODBC データソースサブキーの下に値として表示されないことです。
