# <a name="types-of-data-sources"></a>データ ソースの種類
データ ソースには、マシン データ ソースとファイル データ ソースの 2 種類があります。 どちらもデータのソースに関する同様の情報を含んでいますが、この情報の格納方法は異なります。 これらの違いのために、それらは多少異なる方法で使用されます。

-   [マシン データ ソース](../../odbc/reference/machine-data-sources.md)  
  
-   [[ファイル データ ソース]](../../odbc/reference/file-data-sources.md)
