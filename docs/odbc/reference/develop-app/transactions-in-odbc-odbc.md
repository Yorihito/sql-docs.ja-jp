# <a name="transactions-in-odbc-odbc"></a>ODBC のトランザクション (ODBC)
ODBC でのトランザクションは、接続レベルで完了します。つまり、アプリケーションがトランザクションを完了すると、その接続のすべてのステートメントハンドルを通じて実行されたすべての処理がコミットまたはロールバックされます。
