# <a name="batch-processing"></a>バッチ処理
SQL ステートメントのバッチのサポートは広くありません。そのため、相互運用可能なアプリケーションでは、条件付きで、またはまったく使用しないようにする必要があります。 詳細については、「[バッチの実行](../../../odbc/reference/develop-app/executing-batches.md)」を参照してください。
