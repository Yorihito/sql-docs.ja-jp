> [!NOTE]  
>  このトピックには、Visual FoxPro ODBC ドライバー固有の情報が含まれています。 この関数の一般的な情報については、「 [ODBC API リファレンス](../../odbc/reference/syntax/odbc-api-reference.md)」の該当するトピックを参照してください。  
  
 サポート: 完全  
  
 ODBC API の準拠: レベル1  
  
 ステートメントの実行時に、アプリケーションからドライバーにパラメーターまたは列のデータを送信できるようにします。  
  
 詳細については、 *ODBC プログラマーリファレンス*の「 [sqlputdata](../../odbc/reference/syntax/sqlputdata-function.md) 」を参照してください。
