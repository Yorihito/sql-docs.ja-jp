# <a name="native-error"></a>ネイティブ エラー
プラスのネイティブエラーコードは、デスクトップデータベースドライバーによって生成されます。 Microsoft Jet によって、負のエラーコードが生成されます。 詳細については、 *『 Microsoft Jet データベースエンジンプログラマーズガイド』* を参照してください。
