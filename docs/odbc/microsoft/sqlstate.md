データソースで発生したエラーの場合、ODBC ドライバーは返されたネイティブエラーを適切な SQLSTATE にマップします。 ドライバーまたはドライバーマネージャーによって検出されたエラーについては、ODBC ドライバーまたはドライバーマネージャーによって、適切な SQLSTATE が生成されます。
