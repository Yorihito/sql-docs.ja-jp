# <a name="read-only-status-text-file-driver"></a>読み取り専用の状態 (テキスト ファイル ドライバー)
テキストドライバーが使用されている場合、データソーステーブルは読み取り専用です。 ただし、アプリケーションでは、テキストテーブルの挿入を実行できます。
