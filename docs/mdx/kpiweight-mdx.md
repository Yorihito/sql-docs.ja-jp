# <a name="kpiweight-mdx"></a>KPIWeight (MDX)


  指定された主要業績評価指標 (KPI) の重みを返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
KPIWeight(KPI_Name)  
```  
  
## <a name="arguments"></a>引数  
 *KPI_Name*  
 KPI の名前を指定する有効な文字列式です。  
  
## <a name="remarks"></a>Remarks  
 返される値は、KPI が親に与える影響を示します。
