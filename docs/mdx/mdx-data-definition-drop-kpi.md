# <a name="mdx-data-definition---drop-kpi"></a>MDX データ操作 - DROP KPI


  指定されたキューブから、指定された主要業績評価指標 (KPI) を削除します。  
  
## <a name="syntax"></a>構文  
  
```  
  
DROP KPI CURRENTCUBE | Cube_Name.KPI_Name   
```  
  
## <a name="arguments"></a>引数  
 *Cube_Name*  
 キューブ名を指定する有効な文字列です。  
  
 *KPI_Name*  
 削除する KPI の名前を指定する有効な文字列です。
