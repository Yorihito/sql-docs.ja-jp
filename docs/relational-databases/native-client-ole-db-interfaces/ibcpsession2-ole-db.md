# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

  IBCPSession2 インターフェイスは、IBCPSession の拡張機能です。この機能は、各列に対して IBCPSession::BCPColFmt を呼び出す代わりに使用できるメンバー関数を提供します。  IBCPSession2 は IBCPSession から継承し、新しいメソッドを[IBCPSession2:: BCPSetBulkMode](../../relational-databases/native-client-ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md)1 つ追加します。
