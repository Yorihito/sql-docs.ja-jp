# <a name="protocolname-property-servernetworkprotocolproperty-class"></a>ProtocolName プロパティ (ServerNetworkProtocolProperty クラス)

  参照されたプロパティを所有するプロトコルの名前を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
  
object.PropertyName [= value]  
```

 *object*
