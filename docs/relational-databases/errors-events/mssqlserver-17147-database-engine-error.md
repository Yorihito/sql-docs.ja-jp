## <a name="details"></a>詳細  
  
| 属性 | 値 |  
| :-------- | :---- |  
|製品名|SQL Server|  
|イベント ID|17147|  
|イベント ソース|MSSQLSERVER|  
|コンポーネント|SQLEngine|
