# <a name="rename-a-database-diagram-visual-database-tools"></a>データベース ダイアグラムの名前変更 (Visual Database Tools)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
オブジェクト エクスプローラーでデータベース ダイアグラムの名前を変更できます。 データベースに対する ALTER 権限が必要です。  
  
### <a name="to-rename-a-database-diagram"></a>データベース ダイアグラムの名前を変更するには  
  
1.  オブジェクト エクスプローラーで、 **[データベース ダイアグラム]** フォルダーを展開します。  
  
2.  名前を変更するデータベース ダイアグラムを右クリックします。  
  
3.  ショートカット メニューの **[名前の変更]** をクリックします。  
  
    オブジェクト エクスプローラーにダイアグラム名を囲むエディット ボックスが表示されます。  
  
4.  データベース ダイアグラムの新しい名前を入力し、 **Enter**キーを押します。  
  
    新しい名前のデータベース ダイアグラムが、オブジェクト エクスプローラーに表示されます。
