# <a name="clustered-property-adox"></a>Clustered プロパティ (ADOX)
インデックスがクラスター化されているかどうかを示します。  
  
## <a name="settings-and-return-values"></a>設定と戻り値  
 **ブール**値を設定して返します。  
  
## <a name="remarks"></a>解説  
 既定値は **False**です。  
  
 このプロパティは、既にコレクションに追加されている [インデックス](./index-object-adox.md) オブジェクトでは読み取り専用です。
