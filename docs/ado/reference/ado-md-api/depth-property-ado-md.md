# <a name="depth-property-ado-md"></a>Depth プロパティ (ADO MD)
階層の [レベル](../../../ado/reference/ado-md-api/level-object-ado-md.md) とルートの間のレベル数を示します。  
  
## <a name="return-values"></a>戻り値  
 は整数を返し、は読み取り専用です。  
  
## <a name="remarks"></a>解説  
 階層のルートにある **レベル** の **深さ** の値がゼロ (0) です。
