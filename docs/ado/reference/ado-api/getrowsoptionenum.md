# <a name="getrowsoptionenum"></a>GetRowsOptionEnum

|定数|値|説明|  
|--------------|-----------|-----------------|  
|**adGetRowsRest**|-1|[GetRows](./getrows-method-ado.md)メソッドの*Start*パラメーターによって指定された現在の位置またはブックマークから、**レコードセット**内の残りのレコードを取得します。|
