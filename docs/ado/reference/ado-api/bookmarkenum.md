# <a name="bookmarkenum"></a>BookmarkEnum
操作の開始位置を示すブックマークを指定します。  
  
|定数|[値]|説明|  
|--------------|-----------|-----------------|  
|**adBookmarkCurrent**|0|現在のレコードから開始します。|  
|**adBookmarkFirst**|1|最初のレコードから開始します。|  
|**adBookmarkLast**|2|最後のレコードから開始します。|
