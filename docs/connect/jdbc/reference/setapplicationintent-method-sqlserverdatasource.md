# <a name="setapplicationintent-method-sqlserverdatasource"></a>setApplicationIntent メソッド (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  **applicationIntent** 接続プロパティの値が設定されます。  
  
## <a name="syntax"></a>構文  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>パラメーター  
 *applicationIntent*  
  
 **applicationIntent** 接続プロパティの新しい値。  
  
## <a name="remarks"></a>解説  
 **applicationIntent** 接続プロパティの詳細については、「[接続プロパティの設定](../../../connect/jdbc/setting-the-connection-properties.md)」を参照してください。
